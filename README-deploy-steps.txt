PUBLISH - NETLIFY (free, easiest)
1. Sign in at netlify.com
2. Drag this WHOLE folder onto the deploy drop zone
3. Live in ~30 seconds; add domain in Site settings > Domain management

PUBLISH - GODADDY cPANEL
1. godaddy.com > Web Hosting > Manage > cPanel > File Manager > public_html
2. Back up anything already there, then upload ALL these files
